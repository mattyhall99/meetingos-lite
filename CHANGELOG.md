# Changelog

All notable changes to MeetingOS Lite will be documented in this file.

## [1.0.0] - 2026-05-29

### Added
- Initial release
- TL;DR generation (2-3 sentences)
- Actions table extraction with Owner and Due columns
- Gmail search for Gemini meeting notes
- Google Drive search for Gemini-generated docs
- Uploaded file support (text, markdown, PDF, docx)
- Pasted text support
- Edge case handling (short meetings, no actions, multiple results, non-English)
- Quality checklist enforcement (specific actions, no vague entries, owner required)
